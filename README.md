## PROGETTO PAGINA WEB RESPONSIVE COMPLETA
Il progetto presenta una pagina web professionale e completa, fa riferimento ad un'azienda generica e il tema trattato è la sostenibilità aziendale.
È stata scelta una palette cromatica adatta al contesto sostenibile insieme ad una tipografia coerente e ben leggibile.

# Obiettivo del progetto
* Dimostrazione dell'adattabilità responsiva alle diverse risoluzioni di schermi.

# Introduzione
Il codice sorgente dei file HTML e CSS è opportunamente commentato per favorire la comprensione e la leggibilità da parte di chi consulta il repository.

# Tecnologie utilizzati
* HTML5
* CSS3
  - Flexbox
  - Grid System
  - Responsive Design

# Struttura del progetto
* index.html : file principale della pagina web.
* style.css : foglio di stile associato.
* IMAGES/ : cartella contenente le immagini utilizzate.
* File PDF simulati dei report di sostenibilità.
